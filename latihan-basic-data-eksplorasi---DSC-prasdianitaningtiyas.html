IMPORT DATA SET

data("mtcars")
df <- data.frame(mtcars)

Data set yang digunakan untuk latihan kali ini adalah data set mtcars, dimana data tersebut diambil dari database software R. Data set mtcars berisi tentang spesifikasi dari beberapa varian mobil. Data diambil dari majalah Motor Trend US 1974, dan terdiri dari konsumsi bahan bakar dan 10 aspek desain dan kinerja mobil untuk 32 mobil (model 1973-1974).

Data set terdiri dari 32 observasi atau pengamatan dan terdiri dari 11 variabel. Berikut keterangan dari 11 variabel tersebut: mpg : Miles/(US) gallon cyl : Number of cylinders disp: Displacement (cu.in.) hp : Gross horsepower drat: Rear axle ratio wt : Weight (1000 lbs) qsec: 1/4 mile time vs : Engine (0 = V-shaped, 1 = straight) am : Transmission (0 = automatic, 1 = manual) gear: Number of forward gears carb: Number of carburetors

Visualisasi data

Contoh scatter plot

plot(df$disp)

Dari scatter plot tersebut nampak bahwa pola data menyebar.

basic plot both => point and line

plot(df$mpg, type='b')

berdasarkan plot diatas menunjukkan bahwa pada variabel mpg observasi yang memiliki nilai tertinggi yaitu pada observasi ke 20 (varian mobil Toyota Corolla).

memampilkan garis rata-rata

plot(df$mpg, type = 'b')
lines(seq(1:nrow(df)),
      rep(mean(df$mpg),32),lwd = 2,col='navy')

menampilkan plot x=hp dan y= mpg

plot(df$hp,df$mpg, pch=19)

menampilkan hubungan antar 2 variabel

plot(df)

plot(df$hp, xlab = 'hp', ylab = 'No of Instances', main = 'hp', col = 'blue')

bar plot

horizontal bar plot

barplot(df$hp, main = 'hp',xlab = 'hp', col= 'maroon',horiz = TRUE)

berdasaran grafik tersebut dapat diketahui bahwa varian mobil yang memiliki nilai terbanyak untuk variabel hp yaitu Maserati Bora (observasi ke 31).

vertikal bar plot

barplot(df$hp, main = 'hp',xlab = 'hp', col= 'maroon',horiz = FALSE)

histogram

hist(df$hp)

boxplot

boxplot(df$hp)

boxplot(df$mpg)

berdasarkan grafik boxplot untuk variabel hp dapat diketahui bahwa terdapat data yang outlier yaitu sebanyak 1 observasi. sedangkan grafik boxplot untuk variaebl mg tidak terdapat data yang outlier.

boxplot(df[,0:5], main = 'Multiple Box plot')
